# Übersicht der Anwendungsfälle

Anwendungsfälle (AwF) beschreiben, **wozu** Modelle und Daten erstellt und
genutzt werden. Jeder AwF wird als eigene Seite nach der
[Vorlage](./vorlage-anwendungsfall) dokumentiert.

| Nr. | Anwendungsfall | Status |
| --- | --- | --- |
| AwF-01 | Bestandserfassung und Grundlagenmodelle | Entwurf |
| AwF-02 | Modellbasierte Koordination und Kollisionsprüfung | Entwurf |
| AwF-03 | Mengenermittlung aus Modellen | geplant |

:::tip Neuen AwF anlegen
Kopie der Vorlage als neue Datei `awf-XX-kurztitel.md` in diesem Ordner
anlegen, ausfüllen und per Pull Request einreichen.
:::
