# Begriffe und Abkürzungen

| Begriff | Bedeutung |
| --- | --- |
| **BIM** | Building Information Modelling – Methode zur digitalen Erstellung und Nutzung von Bauwerksinformationen über den Lebenszyklus |
| **AwF** | Anwendungsfall – definierter Zweck, für den Modelle und Daten erstellt oder genutzt werden |
| **CDE** | Common Data Environment – gemeinsame Datenumgebung als einzige verbindliche Informationsquelle im Projekt |
| **IFC** | Industry Foundation Classes – offenes, herstellerneutrales Austauschformat für Bauwerksmodelle (ISO 16739) |
| **IDS** | Information Delivery Specification – maschinenlesbare Spezifikation von Informationsanforderungen an IFC-Modelle |
| **LOIN** | Level of Information Need – Festlegung des Informationsbedarfs je AwF, Meilenstein und Akteur (EN 17412) |

:::note
Diese Liste ist ein Startpunkt und wird projektbegleitend ergänzt.
:::
