# Rollen und Verantwortlichkeiten

| Rolle | Kernaufgaben |
| --- | --- |
| **Auftraggeber (ASTRA)** | Definiert Informationsanforderungen, genehmigt den BIM-Abwicklungsplan, nimmt Liefergegenstände ab |
| **BIM-Gesamtkoordination** | Koordiniert die Fachmodelle, führt die Koordinationssitzungen, verantwortet die modellbasierte Qualitätssicherung |
| **BIM-Koordination Fachbereich** | Stellt die Qualität des eigenen Fachmodells sicher, liefert termingerecht in die CDE |
| **Modellautorinnen und -autoren** | Erstellen und pflegen die Fachmodelle gemäss Modellierungsgrundsätzen |
| **Informationsmanagement (CDE)** | Betreibt die gemeinsame Datenumgebung, verwaltet Zugriffe, Status und Freigabeprozesse |

:::note
Die konkrete Ausgestaltung der Rollen erfolgt projektspezifisch im
BIM-Abwicklungsplan (BAP).
:::
